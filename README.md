## プロジェクトの作成

Gitからクローンしたディレクトリーでターミナルを開いて下記コマンドを実行し仮想環境を作成する

```
$ virtualenv -p python3 venv
$ source venv/bin/activate
```

次にプロジェクトを作成
```
$ pip install django==2.0.4 djangorestframework==3.8.2 channels==2.1.0
$ django-admin startproject [ProjectName]
```
